## <span id="project-overview">ğŸ§© Project Overview</span>

The **Excel Road Accident Analysis Dashboard** provides a **centralized analytical view of road accident casualties** across multiple conditions such as severity, vehicle type, road characteristics, lighting, and area type.

The dashboard helps stakeholders **identify high-risk patterns**, compare trends across years, and support **data-driven road safety decisions**.

---

## <span id="purpose-of-the-dashboard">ğŸ“Œ Purpose of the Dashboard</span>

This dashboard is created to:

- Analyze accident **severity levels**
- Identify **high-risk vehicle types**
- Understand impact of **road type & road surface**
- Compare **urban vs rural accident distribution**
- Study accidents under **daylight vs darkness**
- Track **year-wise and monthly trends**

---

## <span id="dashboard-overview">ğŸ“Š Dashboard Overview</span>

The dashboard consists of:

- KPI cards with severity distribution
- Vehicle-type casualty summary using icons
- Monthly trend comparison (CY vs PY)
- Road and surface-based analysis
- Area-wise and light-condition breakdown
- Interactive filters for dynamic analysis
